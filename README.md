# Todo Evolution - Phase I

A command-line Todo application built with Python.

## Phase I Features
- **In-Memory Storage**: No database or files required.
- **CLI Interface**: Manage tasks via terminal input.
- **Core CRUD**: Add, View, Update, Delete, and Toggle Completion.

## Requirements
- Python 3.10+

## Usage
Run the application from the root directory:
```bash
python src/main.py
```
